# Icons

To add or change an icon add the file to `public/app/frontend/src/assets/icons`, then go to `public/app/frontend/style.scss` and add the file name and extension to the `$iconTypes` object.

Update the Icon Gallery component in `public/app/frontend/documentation/design/Icons.mdx` to include the new icon and its class.

To include an icon in your component use `<i class="icon-{name}--{em|xs|sm|md|lg}">`.  
`em` size is useful for inline icons, it uses 1em for relative sizing.
