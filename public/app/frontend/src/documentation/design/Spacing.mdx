Spacing units are used to create consistent padding and margins throughout the application. They're only used for
spaces: margins, padding and gaps between elements. When setting dimensions (e.g. width and height) you can use any
unit for the design.

To use the spacing units above you can use the following mixin. The mixin takes three arguments: the type of spacing,
the spacing unit and optionally the direction. If you don't include a direction it will default to all sides.

```
@include spacing('margin', 4, 'top');
```
