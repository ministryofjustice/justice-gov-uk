Breakpoints are used to define the width at which a layout changes. For this project we've defined three screen sizes -
mobile, tablet and desktop. Media queries using the preset breakpoints should use `min-width`, for example:

```
.heading {
    font-size: 1.5rem;
    @media screen and (min-width: $justice-lg) {
        font-size: 3rem;
    }
}
```
