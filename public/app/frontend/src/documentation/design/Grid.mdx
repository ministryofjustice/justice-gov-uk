# Grid

Most pages will consist of a header, a menu, the page content and a footer. Each of these sections consists of a container, grid and grid row.

A page using the _with sidebar_ template might look like this on desktop:

<Unstyled style={{ background: '#d7d7d7', border: '1px black solid' }}>
    <div style={{ ...wrapperStyle, background: '#7dadb9' }}>
        <div style={{ ...containerStyle, background: '#7dadb9' }}>
            <div style={{ ...componentStyle }}>Header</div>
        </div>
    </div>
    <div style={{ ...wrapperStyle, background: '#7dadb9' }}>
        <div style={{ ...containerStyle, background: '#7dadb9' }}>
            <div style={{ ...componentStyle, height: '2rem' }}>Menu</div>
        </div>
    </div>
    <div style={{ ...wrapperStyle }}>
        <div style={{ ...containerStyle }}>
            <div style={{ ...gridStyle }}>
                <div style={{ ...gridRowStyle, gridColumn: '1 / span 8' }}>
                    <div style={{ ...componentStyle, height: '6rem' }}>
                        First component
                    </div>
                    <div style={{ ...componentStyle, height: '6rem' }}>
                        Second component
                    </div>
                </div>
                <div
                    style={{
                        ...componentStyle,
                        gridColumn: '9 / span 4',
                        height: '6rem',
                    }}
                >
                    Sidebar
                </div>
            </div>
        </div>
    </div>
    <div style={{ ...wrapperStyle, background: '#7dadb9' }}>
        <div style={{ ...containerStyle, background: '#7dadb9' }}>
            <div style={{ ...componentStyle, marginBottom: '0' }}>Footer</div>
        </div>
    </div>
</Unstyled>

All of the page content is constrained to the maximum width of the page, however by working in rows we're able to keep the content width consistent while full width elements like the header and menu can have their background 'break out' of the maximum width.

## Mobile

**Columns:** 4

**Grid gap:** 1rem

**Gutter:** 1rem

**Maximum width:** 100%

<Unstyled>
    <div
        class="container"
        style={{ background: '#d7d7d7', width: '412px', padding: '0 1rem' }}
    >
        <div
            class="grid"
            style={{
                gridTemplateColumns: 'repeat(4, [col-start] 1fr)',
                gap: '1rem',
            }}
        >
            {[...Array(4).keys()].map(() => {
                return (
                    <div
                        class="grid-row"
                        style={{
                            width: '100%',
                            height: '100px',
                            background: '#5a5c92',
                        }}
                    ></div>
                );
            })}
        </div>
    </div>
</Unstyled>
